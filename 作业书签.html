第一章

  1. 首页命名为index.html
  2. WWW是一个基于超文本方式的信息检索工具, www成功在于它制定了一套标准的,易于人们掌握的超文本开发语言HTML、信息资源的统一定位格式URL和超文本传输协议HTTP
  3. 网络运行:服务器--域名/ip地址--站点--index.html(首页)
    站点通常包含:
    1. html结构层
    2. css表现层
    3. js行为层
    4. img
    1. HTML:超文本标记语言，是构成WEB页面的重要工具
    2. xml:可扩展文本标记语言(区分大小写,自己定义的地方要用双标记)
    3. xhtml:可扩展超文本标记语言 文档类型：框架型 严格型 过渡型
      (现在学的HTML4.0 主要用于PC段' 之后会学H5用于移动端 两者区别H5新增加了语义化标签 用<>来嵌套)
    4. URL（统一资源定位器）用于描述internet上资源的位置和访问方式——包含URI
      格式： 协议名：//网址或IP端口号/URI
    5. URI(统一资源标识符)
      构成：
      1. host（主机名）：定义此域中的主机
      2. path（路径）：定义服务的端口号，端口号通常是被省略的，HTTP默认端口号是80
      3. filename（文件名）：定义的文档名称
  5. 1.http超文本传输协议
    2.smtp邮件传输协议
    3.ftp文件传输协议
  6. 动态页面和静态页面主要不同在于：
    1、静态页面可以不用传输到服务器上；
    2、静态页面不能进行交互
  7. 网页浏览器是显示网页服务器或档案系统内的文件，并使用用户与这些文件进行交互的一种软件。它用来显示万维网或局域网文字、影像及其他资讯

第二章

  1. 在基本的HTML文档中，只需要两种元素存在：一种是HTML标记，另一种则是普通文本
    一个简单的HTML文档的组成结构如下：
    <html> <!--超文本文件开始--> <head> <!--首部元素开始--> <title>文档标题</title> </head> <!--首部元素结束--> <boby> <!--正文元素开始--> 文档部分，正文部分 </boby> <!--正文元素结束--> </html> <!--超文本文件结束-->
  2. HTML文档的head是一个容器元素，在head容器元素中允许出现以下元素：
    1. title元素: <title>文档标题</title>
    2. link元素: <link name="text" rel="text" href="URL">
    3. meta元素 meta是第一个学到的单标记 属性name http-eguiv <meta name="text" content="text"> key="value"属性等于值
    4. style元素：用于写css
    5. script元素：用于写js
    6. meta 单标记
      属性： 1.name 主要用于描述页面

      keywords:关键字

      2.http-equiv 主要用于和服务器相关信息

      refresh:刷新

      content-type:文本类型

      3.（content 是解析这两种属性的）
      4.charset定义字符集 1.gb2312 2.utf-8
    7. body容器属性:
      1. 背景属性:bgcolor背景色 background背景图案
        2. <body background="../img1.jpg"></body>
      2. 文字属性:text正文文字颜色 link链接文字颜色

第三章

  1. 标题文字
    <h#>文字</h#> ..#表示：1-6（数字小文字大） 基本语法： <h#align="left/center/right">标题文字 </h#> align表示标题水平对齐方式 例如： <h1 align="center">朗科网络</h1> h标记是一个段落标记自己单独站一行
  2. 文字标记
    <font face="" size="" color="">文字</font>
    1. face表示文字字体；
    2. size表示文字大小；
    3. color表示文字颜色；
    例如： <font face="微软雅黑" size="-2" color="#f00">好好学习</font> 注释：默认字体3 字体从1-7
  3. 特殊字符:
    常见的特殊字符 空格:&nbsp;--添加空格 版权符:&copy; 大于号:&gt; 小于号:&lt; 注册商标:&reg;
  4. 标记的嵌套
    注意当标记不冲突时全都显示，当有冲突时按就近原则显示
    例如： <h1 align="center"> <font fase="宋体＂color=" #0f0">您好</font> </h1>
  5. 文字修饰
    基本语法：<b>...</b>粗体<strong>...</strong> <i>..</i>斜体<em>...</em> <u>...</u>下划线 <del>...</del>删除线 <sup>...</sup>上标 <sub>...</sub>下标 <tt>..</tt>等宽文字 称为行内标记 <address>....</address>网址 称为块级标记
  6. 段落标记
    基本语法：
    <p>文字内容</p> 块级标记 换行标记：<br> 它是一个单标记 注意：一个<p>空一行，一个</P>空一行 div就是一个单独的块级标记 span就是单独的一个行内标记
  7. 水平分割线
    基本用法： <hr width="" size="" color="" noshade> 它是一个单标记
    1. width表示分割线的宽度；
    2. size表示分割线的粗细；
    3. color表示分割线的颜色；
    4. noshade表示是否有阴影；
    例子： <hr width="500" color="#00" size="100">
  8. 其他段落标记
    段落缩进： <blockquote>文字</blockquote>

第四章

  1. 列表简介
    列表类型及其标记符号
    1. 无序列表：ul
    2. 有序列表：ol
    3. 定义列表：dl
    4. 目录列表：dir
    5. 菜单列表：menu
  2. 无序列表标记
    <ul type=符号类型> <li>第一个列表项</li> </ul> li里面也可以加属性type type属性取值有
    1. disc(实心圆）默认就是实心圆
    2. circle（空心圆）
    3. square(实心方块）
  3. 有序列表标记
    <ol> <li>第一个列表项</li> </ol> start设置起始数位
    使用有序列表标记的type属性取值以及相对应的编号样式如下：
    1. “1”指定项目标号为阿拉伯数字；
    2. “a”指定项目标号为小写英文字母；
    3. “A”指定项目标号为大写英文字母；
    4. “i”指定项目标号为小写罗马数字；
    5. “I”指定项目标号为大写罗马数字；
  4. 定义列表标记
    <dl> <dt>需要解释词</dt> <dd>对解释词的解释说明</dd> </dl>
  5. 目录列表标记 <dir> <li>第一个列表项</li> <li>第二个列表项</li> </dir>
  6. 菜单列表标记
    <menu> <li>第一个列表项</li> <li>第二个列表项</li> </menu>

第六章

  1. 链接路径：
    1. 绝对路径（从站点开始）；
    2. 相对路径；
    3. 物理路径（从磁盘开始）；
    4. 根路径
  2. 创建超链接
    1. 建立超链接的标记用a。以<a>开始，以</a>结束；
    2. 基本语法：<a href="资源地址" title="指向链接显示的文字" target="窗口名称">超链接内容</a>
    3. _biank:在新窗口中打开被链接文档。
  3. 邮件链接:在html页面建立E-mail链接，用户单击链接，系统会自动启动后默认的电子邮件软件，打开一个邮件窗口。
    基本用法： <a href="E-mail地址？subject=邮件主题">描述文件</a>
    1. cc=E-mail地址--抄送发件人
    2. bcc=E-mail地址--暗送发件人
    3. body=邮件内容
  4. 图片链接
    可以为一个图片制定链接 <a href="url" target="目标窗口的打开方式"><img src="图片地址"></a> 例如： <a href="url" target="_blnk"> <img src="url"> </a>
  5. 下载文件链接 <a href="url">链接内容</a>
  6. 书签链接
    在同一页面内要使用书签链接的格式： <a href="#书签名称" target="窗口名称">链接标题</a> 在不同页面之间要使用书签的格式（在不同页面中链接的前提是需要指定好链接的页面地址和链接的书签名称）： <a href="url地址#书签名称" target="窗口名称">链接标题</a> 以上两种链接形式，链接的目标为： <a name="书签名称">链接内容</a> 例如： <body> <a name="top">第一章</a> <a href="#top">返回顶部</a> </body>
  7. 表格
    1. 表格标记
      1. 表格标记 <table></table>
      2. 表头 <th></th> 剧中加粗
      3. 每一行用 <tr></tr>
      4. 每一列用 <td></td>
      5. 标题 <caption></caption>
    2. 表格的整体外观用 <table></table> 标记属性决定
      1. align:定义网页中表格的位置
      2. border:定义表格边框粗细,N取整数,单位是像素
      3. width:定义表格宽度
      4. height:定义表格高度
      5. bgcolor:背景颜色
      6. backgrond:背景图
      7. bordercolor:边界颜色
    3. 表格的背景颜色和尺寸设置
      用bgcolor表示格式的背景设置,格式为: <table bgcolor="#f00">为表格设置红色背景</table>
    4. 表格的宽度和高度设置 <table width=n1 height=n2></table> (可用百分比,可用像素)
  8. 多列合并
    在表格中使用colspan进行跨多列的设置,格式为: <td colspan="n"></td> 说明： 　　colspan="n"中的N表示跨多少列
    例如: <td colspan="3"> </dt>
  9. 多行合并
    在表格中使用rowspan进行跨多行的设置,格式为: <td rowspan="n"></td> 说明： 　　rowspan="n"中的N表示跨多少行
    例如: <td rowspan="3"> </dt>
  10. 对齐列表
    1. 水平对齐的方式是用标记、 、和的align属性,align的属性值分为：center,left,right.
    2. 垂直对齐的方式用valign属性，vglign属性分为：top(靠单元格项)、bottom(靠单元格底)、middle(靠单元个中)
  11. 设置单元格间距
    语法： <table width="" border="1" cellspacing="0"> <tr> <td>..</td> </tr> </table> <table></table> 标记中cellspacing属性可以设置表格中单元格的间距
  12. 设置单元格边距
    在网页文件中，单元格的边距指单元格中的内容与单元格边框的距离
    基本语法： <table width="380" border="1" cellpadding="0"></table>

第七章

  1. 插入图片标记 <img src="url"> 说明：
    1. -img是插入图片的标记，其中src是必须的属性
    2. -src属性用来指定图片源，及图片的URL。
    3. width:指定的图像显示宽度
    4. height：指定图像的显示高度
    5. hspace:定义图像左侧和右侧的空白
    6. vspace:定义图像的顶部底部的空白
    7. align:指定图片的对齐方式
    8. bordr:指定图片的边框大小
    9. alt:如果图片无法显示，代替图片的文字说明
    10. title:鼠标划上图片显示的内容
    11. 注意:高版本浏览器中图片不带边框,低版本浏览器会带有边框,为了兼容性管理 手动加上border="0";
  2. 指定图片的对齐方式<img src="URL" align=""> align取值
    1. left:在水平方向向上向左对齐
    2. right:在水平方向向上向右对齐
    3. top:图片顶部与同行其他元素顶部对齐
    4. middle:图片顶部与同行其他元素中部对齐
    5. bottom:图片顶部与同行其他元素底部对齐
    6. absbottom:图片底部与同行最低元素的底部对齐
    7. absmiddle:图片底部与同行最大元素的中部对齐
    8. baseline:图片底部与文本基准线对齐
    9. texttop:图片底部与同行最高元素的顶部对齐
  3. 背景音乐的标记
    用法： <bgsond src="URL" loop=""></bgsond> loop指定声音文件循环次数，值为-1或infinte指定循环播放次数无限 <bgsond></bgsond> 只能添加声音文件，而且不能显示和调整播放的控制面板
  4. <embed>标记可以插入各种各样的多媒体，如WMV,MP3,AVI,SWF,MOV,RMVB等格式的多媒体文件。
    还可以采用超连接标记将多媒体文件放在网页上，这样当浏览者点击超链接，浏览器将这个多媒体文件先下载到浏览者的机器硬盘上，在条用相应的播放程序播放
    基本用法： <embed src="url" loop="true/false" autostart="true/false" width="宽" height="高" type="media-type" pluginspage="plugin">
    1. true可循环播放无限次，false只播放1次；false为默认值。
    2. autostart指定媒体文件在下载完成后是否自动播放，
    3. width和height分别指定多媒体的宽和高，值都为0时类似背景音乐，可隐藏面板；
    4. type指定多媒体文件类型；
    5. pluginspage：指定插件页面；
    6. 高版本浏览器中图片不带边框,低版本浏览器会带有边框,为了兼容性管理 手动加上border="0";
